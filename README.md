# Statistical Decision Support - Telco Customer Churn

## Objective

This project applies descriptive and inferential statistical techniques to understand customer churn and support business decision-making.

## Statistical Techniques Used

- Descriptive Statistics
- Mean, Median, Mode
- Variance
- Standard Deviation
- Skewness
- Kurtosis
- Correlation Analysis
- Confidence Interval
- Independent T-Test
- Chi-Square Test
- Probability Analysis

## Key Findings

- Monthly charges significantly influence customer churn.
- Contract type has a statistically significant relationship with churn.
- Long-term contracts reduce churn probability.
- Higher-paying customers are more likely to leave.

## Tools Used

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib
- Seaborn
